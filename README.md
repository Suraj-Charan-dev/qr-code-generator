📱 QR Code Generator

A modern and responsive QR Code Generator Web Application built using HTML, CSS, and JavaScript. It allows users to instantly generate QR codes for any text or URL with a simple and clean user interface.

---

🚀 Features

- ✅ Generate QR codes instantly
- ✅ Supports text, links, and other inputs
- ✅ Input validation (prevents empty input)
- ✅ Responsive design (works on mobile & desktop)
- ✅ Clean and modern UI
- ✅ Lightweight and fast

---

🛠️ Technologies Used

- 👉🏻 HTML5 – Structure
- 👉🏻 CSS3 – Styling & Layout
- 👉🏻 JavaScript (ES6) – Logic & Functionality

⚙️ How It Works

1️⃣ User enters text or URL
2️⃣ Clicks on the "Generate QR Code" button
3️⃣ The app processes the input
4️⃣ Displays the generated QR code instantly
